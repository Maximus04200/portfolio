# Portfolio - Martin Di Nota

## Instructions d'installation
ouvrir un terminal/
npm install puis ensuite 
npm start


## Description de mon projet avec react.js :
Ce projet est mon portfolio personnel créé dans le cadre de ma formation développeur web full stack a l'école CEF. Il présente mes compétences, mes réalisations et mes services.

## bibliothèque utilisé : 
- React.js
- React Router DOM (navigation entre les pages)
- Bootstrap 5 (mise en page et composants)
- Bootstrap Icons (icônes)
- React Helmet (SEO)

## Arborescence de mon projet 
src/
components/ composants réutilisables 
 pages/ une page par route 
 data/ mes fichiers de données centralisées

## Le dossier data/ qui me permet de modifier plus simplement mon code pour l'avenir 

Pour modifier les projets : src/data/projets.js
Pour modifier les services : src/data/services.js  
Pour modifier les compétences : src/data/competences.js

## Mes pages inclut dans le projet 
- Accueil : présentation, compétences, modale GitHub
- Services : mes offres de services
- Réalisations : mes projets pour l'instant aucun 
- Contact : formulaire et carte Google Maps (fausse adresse)
- Mentions légales : informations légales
